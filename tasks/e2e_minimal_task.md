# E2E smoke: минимальный проект

Повторяемый сценарий: одно и то же ТЗ даёт сопоставимый объём работ (минимальный стек).

## Требования

Создай **минимальный** проект на **Python 3**:

- Один файл `main.py` с функцией `main()` и блоком `if __name__ == "__main__": main()`, печатающим строку `e2e_ok`.
- Файл `README.md` с названием проекта `e2e_minimal` и одной строкой описания.
- Файл `requirements.txt` (можно пустой или только комментарий).

Без веб-сервера, БД, Docker внутри генерации и без сложных тестов: если нужны команды проверки — достаточно `python -m py_compile main.py`.
